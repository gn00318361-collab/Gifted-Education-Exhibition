# 種子 — 畢業成果展邀請函設計提案

桃園市青溪國小 114 學年度第九屆創造力資優班畢業成果展

## 展覽資訊

- **展期**：115.05.11（一）— 05.15（五）
- **開幕**：5/11（一）19:00，青溪國小 B1 簡報室
- **展覽會場**
  - 會場一：桃園市土地公文化館（桃園區三民路一段100號）
  - 會場二：青溪國小多功能教室 2F（桃園區自強路80號）

## 設計方案

本頁面包含三款邀請函設計（正反面），供主任參閱選用：

| 版本 | 風格 | 說明 |
|------|------|------|
| Design 01 | 票券撕線 | 仿展覽入場券，右側副券可撕下留念 |
| Design 02 | 圓形窗口 | 圓形窗口搭配種子發芽插圖，優雅留白 |
| Design 03 | 色塊分割 | 左側深色塊＋直排主題字，現代俐落 |

## 種子紙

邀請函擬使用種子紙印製，紙張內含植物種子。展覽結束後，將卡片撕碎埋入土中、適當澆水，即可發芽成長。

## 部署方式

開啟 `index.html` 即可在瀏覽器中預覽。如需產生網址分享：

1. 前往 [Netlify Drop](https://app.netlify.com/drop)
2. 將本資料夾拖曳上傳
3. 即可取得公開網址
