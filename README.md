# Gifted-Education-Exhibition

桃園市青溪國小 114 學年度創造力資優班畢業成果展——「種子」邀請函網頁。

## 專案說明

本專案為第九屆創造力資優班畢業成果展的線上邀請函，以單一 HTML 檔案呈現三種不同風格的邀請卡設計，供評選與印刷使用。

## 檔案結構

```
index.html   # 邀請函主頁（三版比較，含正反面）
README.md    # 本文件
```

## 技術細節

- **純靜態頁面**：單一 `index.html`，無需建置工具或後端伺服器
- **字型**：透過 Google Fonts 載入 Noto Serif TC / Noto Sans TC
- **CSS-only 設計**：所有視覺效果（撕線、圓形窗口、色塊分割）皆以純 CSS 實現，未使用 JavaScript
- **SVG 插圖**：種子發芽圖示以 inline SVG 繪製，不依賴外部圖檔
- **響應式列印**：支援 `@media print`，可直接從瀏覽器列印

## 三種版本

| 版本 | 風格 | 設計特色 |
|------|------|---------|
| 版本一 | 票券撕線 | 仿展覽票券，右側副券可撕下留念 |
| 版本二 | 圓形窗口 | 左側圓形窗口搭配種子發芽插圖 |
| 版本三 | 色塊分割 | 左側深色塊搭配直排主題字 |

每個版本皆包含正面與背面設計，背面統一呈現展覽資訊、會場地址及 QR Code 預留區。

## 部署

本專案透過 GitHub Pages 部署，推送至 `main` 分支後自動上線：

```
https://<username>.github.io/Gifted-Education-Exhibition/
```

如需本地預覽，直接以瀏覽器開啟 `index.html` 即可。
